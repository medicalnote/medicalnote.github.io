HUGOプロジェクト構成

設定ファイル config.toml

/themes
デザインレイアウト
ここにテーマフォルダを入れて書き出すと簡単に確認ができる
テーマによってconfig.tomlの設定をしないとエラーになるものあり

/layouts
*shotocodes*を使って画像,動画,iframeなど設定できる。
その他パーツを作成できそうだがまだ未検証。

記事の作成

    hugo new post/*****.md

    title: "HUGOブログ概要"

  • 確認方法

    hugo server -t minimal -D -w

    ＊　-t テーマ名 ＊　buildはdeploy時
